# id-labor-law-rag-assistant
Asisten AI tanya-jawab hukum ketenagakerjaan Indonesia berbasis Qwen2.5-1.5B, dilatih bertahap (fine-tuning + GRPO reasoning) lalu dipadu RAG atas UU Cipta Kerja dan PP turunannya. Fitur: reasoning transparan, sitasi sumber, hybrid retrieval (BM25+FAISS), HyDE, reranking cross-encoder, fallback pencarian internet, dan UI Gradio.
